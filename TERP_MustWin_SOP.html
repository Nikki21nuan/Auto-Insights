Q2
Sales online 更新项目状态
▼
如果料号 PN 区域是空白的，请随机填写料号信息再进行操作
解决方法：
  1. 点击项目下拉至料号 PN 区域 - 勾选 Select ALL PN
  2. 更新状态，并 Save
▲ 勾选 Select All PN
▲ 更新 Status（Lost / Closed - Customer Cancelled）→ Save
Q3
TERP 系统生成金额和实际计算金额不一致
▼
解决方法：
  1. 点击 Export 导出 PN 表格
  2. 检查表格 E-M 列的数量是否乘 PN usage
举例：项目对应的 IHS 车型预测是 20,000 辆/年，1-1718333-1 单车用量是 2 个，则 E-M 列所有 Qty 为 40,000
计算公式

PN Qty = Vehicle Volume × PN Usage

▲ TERP Billing $1,440,796（wrong）· 点击 Export 导出
▲ 导出的 Excel · 检查 E-M 列 Qty
Q4
在新建项目时, 找不到 PN 批量上传"Template"按钮
▼
解决方法：
  1. 点击 Save as Draft 后重新打开该项目
  2. 点击 "Template" 下载模板 - "Import" 上传模板 - "Export" 导出模板
Template 下载 Import 上传 Export 导出
▲ 步骤 1 · 点击 Save as Draft
▲ 步骤 2 · 重开后显示 Template / Import / Export 三个按钮
